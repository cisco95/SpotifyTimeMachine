# SpotifyTimeMachine

Creates a spotify playlist using a date provided by the user and the Billboard Top 100 website.

### Dependencies:

- Spotipy
- BeautifulSoup

### How It Works:

1.  Run
2.  Enter Date (YYYY-MM-DD)
3.  Log In to Spotify, Agree to Hot 100 Time Machine terms.
4.  Paste URL redirected to.

#### Playlist created!
